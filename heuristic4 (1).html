4. Consistency and Standards

Short Summary

The fourth usability heuristic, Consistency and Standards, means that a system should use familiar design patterns, words, and actions so users don’t have to guess what things mean or how they work. When buttons, icons, and layouts follow common standards, users can easily understand and navigate the interface. For example, using a gear icon for settings or keeping navigation menus in the same place across pages helps create a predictable experience. Consistency builds trust and reduces confusion, allowing users to focus on their tasks instead of learning how the system works each time.
